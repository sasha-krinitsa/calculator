# Консольный калькулятор
=
Консольный калькулятор на Python, который:

● Поддерживает операции: +, -, *, /.

● Сохраняет историю вычислений в список.

● Выводит результат операции.

● По завершении выводит полную историю вычислений.

● Не допускает деления на ноль.

● Имеет переменную gitHub с ссылкой на твой репозиторий на GitHub.

Программа работает в цикле с меню команд.

Требования:

● Программа должна быть написана на Python.

● Все операции обрабатываются через меню.

● Используется список для хранения истории.

● Деление на ноль должно вызывать сообщение: "Ошибка: Деление на ноль."

● Добавьте переменную gitHub в начало или конец программы:

gitHub = "https://github.com/ ваш_логин/ваш_репозиторий"

● Код должен соответствовать PEP8.

● Программа не должна выдавать ошибок при некорректном вводе.

● Проект должен быть загружен в GitHub с использованием git.
